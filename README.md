# CareerPilot AI 🚀

CareerPilot AI is an intelligent AI-powered job search assistant that analyzes resumes, extracts skills, recommends jobs, calculates match scores, improves resumes, and simulates job applications.

## Features

✔ Resume Parsing  
✔ Skill Extraction using NLP  
✔ AI Job Recommendation  
✔ Resume-Job Match Score Calculation  
✔ AI Resume Improvement  
✔ Automated Job Application Simulation  
✔ Streamlit Dashboard Interface  

## Tech Stack

- Python
- FastAPI
- Streamlit
- NLP
- Sentence Transformers
- Machine Learning
- BeautifulSoup (Web scraping)

## System Architecture

Resume Upload  
↓  
Resume Parser  
↓  
Skill Extraction  
↓  
Job Recommendation  
↓  
Match Score Calculation  
↓  
Resume Improvement  
↓  
Auto Job Application
